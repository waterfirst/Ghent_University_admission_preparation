1단원: 미적분학 (Calculus)

출제 빈도 높은 주제: 도함수의 정의와 계산, 접선 방정식, 함수의 증가/감소 및 부호, 무리함수와 절댓값 함수의 성질, 극한과 점근선, 부정적분(치환적분 중심)

1.1 함수의 기본 성질과 그래프

🔑 핵심 개념

  • 정의역(Domain): 함수가 정의되는 x값의 집합
    • 다항함수 → ℝ 전체
    • 유리함수 P(x)/Q(x) → Q(x) ≠ 0
    • 무리함수 √R(x) → R(x) ≥ 0
  • 영점(Zeros): f(x) = 0인 x → 그래프가 x축과 만나는 점
  • 증가/감소: 도함수 f'(x) > 0이면 증가, < 0이면 감소
정의역을 먼저 확인! 무리함수에서는 부등식 풀 때 양변 제곱 전 양수 확인 필수. 그래프 개형보다 부호·증감·영점에 집중하세요.

1.2 절댓값 함수

$|x| = \begin{cases} x & \text{if } x \geq 0 \\ -x & \text{if } x < 0 \end{cases}$

🔍 성질

  • 항상 |x| ≥ 0
  • 미분 불가능한 점 존재 (예: x = 0에서 꺾임)
  • 삼각부등식: |a + b| ≤ |a| + |b|

1.3 일대일 함수와 역함수

🔑 정의

  • 일대일 함수: x₁ ≠ x₂ ⇒ f(x₁) ≠ f(x₂)
  • 가로선 테스트: 수평선이 그래프와 두 번 이상 만나면 일대일 아님
모의고사 6번 분석
f(n,m) = 3^m × 5^n: 소인수분해의 유일성에 의해 서로 다른 (m,n)이면 값이 다름 → 일대일! (정답 C)
f(n,m) = m+n: (1,3)과 (2,2)가 같은 값 → 일대일 아님

1.4 지수·로그함수의 성질

함수 정의역 치역 증감성
a^x (a>1) ℝ (0, ∞) 증가
log_a x (a>1) (0, ∞) ℝ 증가

🔧 도함수

  • $\frac{d}{dx}(e^x) = e^x$
  • $\frac{d}{dx}(\ln x) = \frac{1}{x}$
  • $\frac{d}{dx}(a^x) = a^x \ln a$
  • $\frac{d}{dx}(\log_a x) = \frac{1}{x \ln a}$

1.5 극한과 연속성

🔑 정의

  • lim(x→a) f(x) = L: x가 a에 가까워질 때 f(x)가 L에 가까워짐
  • 연속: lim(x→a) f(x) = f(a)

1.6 점근선 (Asymptotes)

종류 조건 예시
수직 lim(x→a) f(x) = ±∞ f(x)=1/(x-2) → x=2
수평 lim(x→±∞) f(x) = L f(x)=(2x+1)/x → y=2
사선(Oblique) 차수: 분자 = 분모 + 1 f(x)=x²/(x+1) → y = x - 1
유리함수에서 분자 차수가 분모보다 1 크면 사선점근선 존재

1.7 도함수의 정의와 계산

🔑 정의

$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$

🔧 기본 도함수

  • $(x^n)' = nx^{n-1}$
  • $(\sin x)' = \cos x$, $(\cos x)' = -\sin x$
  • $(e^x)' = e^x$, $(\ln x)' = \frac{1}{x}$

🔗 연쇄법칙 (Chain Rule)

$\frac{d}{dx} f(g(x)) = f'(g(x)) \cdot g'(x)$
모의고사 1번 예제
f(x) = x cos(x²)
→ f'(x) = cos(x²) + x · (-sin(x²)) · 2x = cos(x²) - 2x² sin(x²)
x = √(2π)/2일 때 → x² = π/2
cos(π/2) = 0, sin(π/2) = 1
f'(x) = 0 - 2 · (π/2) · 1 = -π (정답 A)
합성함수 → 연쇄법칙 필수!

1.8 도함수의 응용

🔹 접선 방정식

$y = f(a) + f'(a)(x - a)$

🔹 극값

  • f'(x) = 0이면서 부호 변화 → 극대/극소

🔹 오목성 (Concavity)

  • f''(x) > 0 → 위로 볼록 (concave up)
  • f''(x) < 0 → 아래로 볼록 (concave down)

🔹 로피탈 법칙 (L'Hôpital's Rule)

0/0 또는 ∞/∞ 꼴일 때:

$\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$

1.9 부정적분과 적분법

🔑 부정적분 정의

$F'(x) = f(x) \quad \Rightarrow \quad \int f(x)\,dx = F(x) + C$

🔧 기본 적분

  • $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ (단, $n \neq -1$)
  • $\int e^x dx = e^x + C$
  • $\int \frac{1}{x} dx = \ln|x| + C$

🔄 치환적분 (Substitution)

$\int f(g(x))g'(x)\,dx = \int f(u)\,du \quad \text{(단, } u = g(x)\text{)}$
모의고사 4번 예제
∫[e^√x / √x]dx
u = √x ⇒ x = u² ⇒ dx = 2u du
∫(e^u / u) · 2u du = ∫2e^u du = 2e^u + C = 2e^√x + C (정답 B)

🧩 부분적분 (Integration by Parts)

$\int u\,dv = uv - \int v\,du$

2단원: 대수학 (Algebra)

출제 포인트: 다항식 나눗셈(조립제법) → 나머지 구하기, 이항정리 → 특정 항의 계수, 다양한 함수를 포함한 방정식/부등식 풀이 → 정의역 체크 필수!

2.1 다항식의 나눗셈

🔑 핵심 개념

  • 나머지정리: f(x)를 (x - a)로 나눈 나머지는 f(a)
  • 조립제법: 일차 인수 (x - a)로 빠르게 나눌 때 사용
f(x) = 2x³ - 3x² + x - 5를 (x - 2)로 나누면:
f(2) = 2(8) - 3(4) + 2 - 5 = 16 - 12 + 2 - 5 = 1 → 나머지 = 1
조립제법은 계수가 정수이고 인수가 일차일 때 유용. 복잡한 나눗셈은 나머지정리로 빠르게 검증 가능.

2.2 이항정리 (Binomial Theorem)

🔑 정의

$(a + b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k} b^k$
  • $\binom{n}{k} = \frac{n!}{k!(n-k)!}$: 이항계수
  • 특정 항: $(k+1)$번째 항 = $\binom{n}{k} a^{n-k} b^k$

📌 자주 묻는 질문 유형

(x + 2)⁵에서 x³의 계수는?
→ k = 2 (왜냐면 x^(5-2) = x³)
→ C(5,2) · 2² = 10 · 4 = 40
지수와 항 번호의 관계를 정확히 파악해야 함. a = x, b = 1/x 같은 경우, 상수항은 x^k · x^(-k) = x⁰

2.3 방정식과 부등식 풀이

📌 공통 원칙

  1. 정의역 먼저 확인
    • 유리식: 분모 ≠ 0
    • 무리식: √ 안 ≥ 0
    • 로그: 진수 > 0, 밑 > 0 and ≠ 1
  2. 양변 제곱 시 무연근 발생 가능성 → 반드시 검증!
  3. 부등식: 로그/무리 함수에서 함수의 증감성 고려

✍️ 유형별 풀이 전략

(1) 유리방정식

2/(x-1) = 3/(x+2)
→ x ≠ 1, -2 확인 후, 양변에 (x-1)(x+2) 곱함
→ 2(x+2) = 3(x-1) ⇒ 2x + 4 = 3x - 3 ⇒ x = 7

(2) 무리방정식

√(x + 3) = x - 1
→ 양변 ≥ 0 → x - 1 ≥ 0 ⇒ x ≥ 1
→ 제곱: x + 3 = (x - 1)² = x² - 2x + 1
→ x² - 3x - 2 = 0 → 해 구한 후 x ≥ 1에 만족하는지 확인

(3) 지수방정식

밑 통일: 4^x = 8 ⇒ (2²)^x = 2³ ⇒ 2x = 3 ⇒ x = 3/2

(4) 로그방정식

log₂(x+1) + log₂(x-1) = 3
→ log₂[(x+1)(x-1)] = 3 ⇒ x² - 1 = 2³ = 8
→ x² = 9 ⇒ x = ±3, but x>1 → x = 3
모든 해는 정의역에 포함되어야 함! 검증 단계 생략 금지.

2.4 연립방정식

🔑 해의 종류

  • 유일한 해: 두 직선이 한 점에서 만남 → 기울기 다름
  • 무수히 많은 해: 두 식이 동일 → 일치
  • 해 없음: 평행 → 기울기 같고 y절편 다름
비선형 연립 예제
x + y = 5, x² + y² = 17
→ y = 5 - x 대입 → x² + (5 - x)² = 17
→ x² + 25 - 10x + x² = 17 ⇒ 2x² - 10x + 8 = 0
→ x² - 5x + 4 = 0 ⇒ (x-1)(x-4) = 0 → (1,4), (4,1)
비선형(2차) 연립은 대입법이 가장 안정적. 해가 없다면 판별식 < 0 확인

3단원: 삼각법 (Trigonometry)

출제 포인트: 도 ↔ 라디안 환산, 삼각함수 정의와 그래프 특성, 핵심 공식 활용, 삼각함수 간 변환 문제, 삼각방정식 일반해 vs 특정 구간 해

3.1 각의 단위: 도수법 vs 호도법

🔑 기본 환산

$180° = \pi \text{ rad} \quad \Rightarrow \quad 1° = \frac{\pi}{180} \text{ rad}, \quad 1 \text{ rad} = \frac{180°}{\pi}$
90° = π/2
2π/3 = 120°
도함수·적분·극한 계산에서는 반드시 라디안 사용! 계산기 사용 시 DEG/RAD 모드 확인 필수

3.2 삼각함수의 정의 및 그래프

🔹 단위원 정의

  • cos θ = x, sin θ = y → 단위원 위 점 (x, y)
  • tan θ = sin θ / cos θ = y/x (cos θ ≠ 0)

🔹 주기성

  • sin(θ + 2π) = sin θ, cos(θ + 2π) = cos θ → 주기 2π
  • tan(θ + π) = tan θ → 주기 π

🔹 그래프 특성

함수 정의역 치역 수직점근선
sin x, cos x ℝ [-1, 1] 없음
tan x x ≠ π/2 + kπ ℝ x = π/2 + kπ

3.3 핵심 삼각함수 공식

🔸 기본 항등식

$\sin^2 x + \cos^2 x = 1$

여기서 파생:

  • $1 + \tan^2 x = \sec^2 x$
  • $1 + \cot^2 x = \csc^2 x$

🔸 덧셈/뺄셈 공식

$\sin(A \pm B) = \sin A \cos B \pm \cos A \sin B$
$\cos(A \pm B) = \cos A \cos B \mp \sin A \sin B$
$\tan(A \pm B) = \frac{\tan A \pm \tan B}{1 \mp \tan A \tan B}$

🔸 2배각 공식

$\sin 2x = 2 \sin x \cos x$
$\cos 2x = \cos^2 x - \sin^2 x = 2\cos^2 x - 1 = 1 - 2\sin^2 x$
$\tan 2x = \frac{2 \tan x}{1 - \tan^2 x}$

🔸 반각 공식

sin²(x/2) = (1 - cos x) / 2
cos²(x/2) = (1 + cos x) / 2
공식을 외우기보다 유도 가능하도록 이해할 것

3.4 삼각항등식 증명 전략

✅ 삼각항등식 풀이 일반 전략

  1. 모든 삼각함수를 sin, cos로 통일
  2. 공통분모, 인수분해, sin² + cos² = 1 활용
  3. 복잡한 쪽 → 간단한 쪽으로 변형
다음과 같은 식을 간단히 하라:
1/(cos x tan x) = 1/(cos x · sin x/cos x) = 1/sin x = csc x

3.5 삼각방정식

🔑 일반해 vs 구간해

sin x = 1/2 → 일반해:
x = π/6 + 2kπ 또는 x = 5π/6 + 2kπ, k ∈ ℤ

구간 [0, 2π) 내 해: x = π/6, 5π/6

📌 유의사항

  • tan x = a → 해는 x = arctan a + kπ
  • 무리수/분수 각에서는 삼각비 외우기:
    • sin 30° = 1/2
    • cos 60° = 1/2
    • tan 45° = 1

3.6 직각삼각형에서의 삼각법

🔺 기본 정의

  • sin θ = 대변/빗변
  • cos θ = 인접변/빗변
  • tan θ = 대변/인접변

4단원: 기하학 (Geometry)

출제 범위: 점, 좌표, 방정식, 원의 방정식, 직선의 방정식, 평행선과 수직선의 조건

4.1 좌표와 기본 개념

🔹 두 점 사이의 거리

점 A(x₁, y₁), B(x₂, y₂) 사이의 거리:

$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$

🔹 중점 좌표

$M = \left( \frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2} \right)$

🔹 기울기 (Slope)

$m = \frac{y_2 - y_1}{x_2 - x_1} \quad (x_1 \neq x_2)$
  • 수직선 → 기울기 정의되지 않음
  • 수평선 → 기울기 0

4.2 직선의 방정식

🔸 기본 형태 정리

형태 식 설명
점-기울기식 y - y₁ = m(x - x₁) 한 점과 기울기 알려졌을 때
기울기-절편식 y = mx + b b: y절편
일반형 Ax + By + C = 0 기울기 = -A/B (B ≠ 0)

🔸 평행과 수직 조건

  • 평행: 기울기 같음 → m₁ = m₂
  • 수직: 기울기 곱 = -1 → m₁ · m₂ = -1
직선 y = 2x + 3에 평행 → 기울기 2
수직 → 기울기 -1/2

4.3 원의 방정식

🔸 표준형 (가장 중요!)

$(x - h)^2 + (y - k)^2 = r^2$
  • 중심: (h, k)
  • 반지름: r > 0

🔸 일반형 → 표준형으로 변환 (완전제곱!)

$x^2 + y^2 + Dx + Ey + F = 0$

→ x, y 항별로 완전제곱:

$x^2 + Dx = \left(x + \frac{D}{2}\right)^2 - \left(\frac{D}{2}\right)^2$
$y^2 + Ey = \left(y + \frac{E}{2}\right)^2 - \left(\frac{E}{2}\right)^2$
모의고사 3번 분석
y² - 2y + x² + 6x - 15 = 0

풀이:
1. x 항 정리: x² + 6x = (x + 3)² - 9
2. y 항 정리: y² - 2y = (y - 1)² - 1
3. 대입:
(x + 3)² - 9 + (y - 1)² - 1 - 15 = 0
⇒ (x + 3)² + (y - 1)² = 25
4. → 중심 M = (-3, 1), 반지름 R = √25 = 5
5. 문제: 2a + b + R² = 2(-3) + 1 + 25 = -6 + 1 + 25 = 20

✅ 정답: (C) 20
일반형 → 반드시 완전제곱! 실수하지 않도록 상수항 이동 후 제곱 보정 단계 명확히!

4.4 기하학 종합 문제 유형

유형 1: 원과 직선의 관계

  • 접선일 조건: 중심에서 직선까지 거리 = 반지름
  • 거리 공식: 점 (x₀, y₀)에서 직선 Ax + By + C = 0까지 거리
$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$

유형 2: 두 원의 관계

중심 간 거리 d, 반지름 r₁, r₂

  • d > r₁ + r₂: 떨어짐
  • d = r₁ + r₂: 외접
  • |r₁ - r₂| < d < r₁ + r₂: 두 점에서 만남
  • d = |r₁ - r₂|: 내접
  • d < |r₁ - r₂|: 포함

4단원 핵심 체크리스트

  • 직선의 기울기, 방정식 형태 자유자재?
  • 두 직선이 평행/수직인지 판단 가능?
  • 원의 일반형을 완전제곱으로 표준형 변환 가능?
  • 중심, 반지름, 2a + b + R² 같은 식 계산 정확?
  • 중점, 거리, 기울기 공식 암기?
