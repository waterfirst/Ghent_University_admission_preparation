## 📚 프로젝트 소개

겐트대학교 글로벌 캠퍼스(GUGC) 온라인 입학시험 대비 **화학 강의록**입니다. 모의고사 10문제(11-20번)를 기반으로 6개 단원으로 구성되었습니다.

## 📖 목차

### 🧪 화학 6개 단원

1. **원자 구조와 주기율표**
   - 원자 번호, 질량수, 중성자 계산
   - 전자 배치 (K, L, M 껍질)
   - 주기율표 경향성 (원자 반지름, 이온화 에너지)

2. **화학 결합과 분자 구조**
   - 이온/공유/금속 결합
   - 혼성 궤도 (sp, sp², sp³)
   - Lewis 구조
   - 극성/비극성 분자

3. **산-염기 반응과 pH**
   - 브뢴스테드-로리 정의
   - pH, pOH 계산
   - 몰농도와 희석 (C₁V₁ = C₂V₂)
   - 중화 반응

4. **산화-환원 반응**
   - 산화수 규칙
   - 산화제/환원제 식별
   - 반쪽 반응식 (MnO₄⁻, I⁻)
   - 전자 이동 계산

5. **양적 관계와 기체 법칙**
   - 몰(mol) 계산
   - STP 조건 (22.4 L/mol)
   - 한계 반응물
   - 이상 기체 법칙 (pV = nRT)

6. **반응 속도와 화학 평형**
   - 충돌 이론
   - 촉매의 역할
   - 활성화 에너지
   - 온도 단위 변환

## 💡 학습 팁

### 필수 암기
- STP: 1 mol = 22.4 L
- pH + pOH = 14
- 희석: C₁V₁ = C₂V₂
- MnO₄⁻ → Mn²⁺: 5e⁻

### 실수 방지
- 단위 변환 주의 (mol, L, g, ton)
- 중성자 = 질량수 - 원자번호
- 혼성화 = 결합수 + 비공유쌍
