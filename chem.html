강의록 소개

이 강의록은 겐트대학교 글로벌 캠퍼스(GUGC) 온라인 입학 시험 화학 파트를 준비하기 위한 핵심 강의록입니다. 모의고사 10문제(11-20번)를 기반으로 6개 단원으로 구성하였으며, 각 단원은 개념→공식→예제 순서로 전개됩니다.

출제 영역 개요

  • 1단원: 원자 구조와 주기율표 - 전자배치, 중성자 계산
  • 2단원: 화학 결합과 분자 구조 - 혼성화, Lewis 구조
  • 3단원: 산-염기 반응과 pH - 중화반응, 희석 계산
  • 4단원: 산화-환원 반응 - 전자 이동, 반쪽 반응식
  • 5단원: 양적 관계와 기체 법칙 - 몰 계산, STP 부피
  • 6단원: 반응 속도와 화학 평형 - 촉매, 활성화 에너지
화학은 공식 암기보다 원리 이해가 중요합니다. 각 단원의 핵심 개념을 먼저 이해한 후 예제를 풀어보세요!

1단원: 원자 구조와 주기율표

출제 포인트: 원자 번호와 질량수로 중성자 계산, 전자 배치 순서, 주기율표의 경향성

1.1 학습 목표

  • 원자의 기본 구성 요소 설명
  • 원자 번호, 질량수, 중성자 수 계산
  • 전자 배치와 전자껍질 이해
  • 주기율표에서 원소의 성질 경향 분석

1.2 핵심 개념

🔸 원자의 구성

  • 양성자(Proton): +1 전하, 원자핵 내부
  • 중성자(Neutron): 0 전하, 원자핵 내부
  • 전자(Electron): -1 전하, 원자핵 주위

🔸 원자 번호와 질량수

원자 번호 (Z) = 양성자 수 = 전자 수 (중성 원자)
질량수 (A) = 양성자 수 + 중성자 수
중성자 수 (N) = A - Z

🔸 전자 배치

보어 모델: 전자는 K, L, M, N… 껍질에 배치

  • K 껍질 (n=1): 최대 2개
  • L 껍질 (n=2): 최대 8개
  • M 껍질 (n=3): 최대 18개
인(P, Z=15)의 전자 배치
K: 2개, L: 8개, M: 5개 → 2, 8, 5
오비탈 표기: 1s² 2s² 2p⁶ 3s² 3p³

1.3 주기율표의 경향성

성질 → 방향 (주기) ↓ 방향 (족)
원자 반지름 감소 증가
이온화 에너지 증가 감소
전기음성도 증가 감소

1.4 모의고사 연계 예제

문제 17번: 원자 번호가 X, 질량수가 Y일 때 중성자 수는?

정답: (A) Y - X

해설:
중성자 수 = 질량수 - 원자 번호 = Y - X
문제 18번: 인(P)의 원자 번호는 15다. 전자 배열은?

정답: (B) 2, 8, 5

해설:
- K껍질: 2개
- L껍질: 8개
- M껍질: 5개
총 15개 전자 → 3주기, 15족 원소

  • 주기율표의 경향성 암기 완료?
  • 동위원소 개념 이해?

2단원: 화학 결합과 분자 구조

출제 포인트: 혼성 궤도 판단(sp, sp², sp³), Lewis 구조, 극성/비극성 분자

2.1 학습 목표

  • 이온 결합, 공유 결합, 금속 결합 구분
  • Lewis 구조 그리기
  • 극성/비극성 분자 판단
  • 혼성 궤도 이해 및 분자 기하 예측

2.2 화학 결합의 종류

결합 종류 형성 원소 특징
이온 결합 금속 + 비금속 전자 이동, 정전기적 인력
공유 결합 비금속 + 비금속 전자 공유
금속 결합 금속 + 금속 전자 바다 모델

2.3 혼성 궤도(Hybridization)

🔸 혼성 궤도 판단 공식

혼성 궤도 수 = σ 결합 수 + 비공유 전자쌍 수

- 2개 → sp (직선형)
- 3개 → sp² (평면 삼각형)
- 4개 → $sp^3$ (정사면체)

🔸 σ 결합과 π 결합

  • σ 결합: 단일 결합, 오비탈 축 방향 겹침
  • π 결합: 이중/삼중 결합의 추가 결합
  • 예: C=O → 1개 σ + 1개 π

2.4 모의고사 연계 예제

문제 19번: CH₃NO₂에서 N의 혼성 궤도는?

정답: (B) sp²

해설:
질소(N) 주위 분석:
- C-N: 1개 σ 결합
- N=O: 1개 σ 결합 (이중결합 중)
- 비공유 전자쌍: 1개
→ 총 3개 영역 → sp² 혼성화
혼성화 문제는 중심 원자를 먼저 찾고, 그 주위의 결합과 비공유 전자쌍을 세는 것이 핵심입니다!

2.5 극성과 비극성

🔸 극성 분자 판단 기준

  1. 전기음성도 차이 존재
  2. 분자 구조가 비대칭
극성 분자: H₂O, NH₃, HF
비극성 분자: CH₄, CO₂, CCl₄ (대칭 구조)

2단원 체크리스트

  • 혼성 궤도 공식 암기 완료?
  • σ와 π 결합 차이 이해?
  • 극성 분자 판단 기준 숙지?

3단원: 산-염기 반응과 pH

출제 포인트: pH 계산, 희석 공식 적용, 중화 반응, 몰농도 변환

3.1 학습 목표

  • 브뢴스테드-로리 산-염기 정의 이해
  • pH, pOH 계산
  • 중화 반응과 적정 개념
  • 농도 단위 변환 및 희석 계산

3.2 산-염기 정의

🔸 브뢴스테드-로리 정의

  • 산(Acid): H⁺ 공여자 (양성자 기부)
  • 염기(Base): H⁺ 수용자 (양성자 수용)
HCl + H₂O → H₃O⁺ + Cl⁻
HCl = 산, H₂O = 염기

3.3 pH와 pOH

pH = $-\\log[\\text{H}^+]$
pOH = $-\\log[\\text{OH}^-]$
pH + pOH = 14 (25°C)

🔸 pH 척도

  • pH < 7: 산성
  • pH = 7: 중성
  • pH > 7: 염기성

3.4 몰농도와 희석

🔸 몰농도(Molarity)

M = $\\frac{\\text{mol}}{\\text{L}}$ = $\\frac{\\text{질량(g)}}{\\text{몰질량(g/mol)} \\times \\text{부피(L)}}$

🔸 희석 공식

$C_1V_1 = C_2V_2$

$C_1$ = 초기 농도, $V_1$ = 초기 부피
$C_2$ = 최종 농도, $V_2$ = 최종 부피

3.5 모의고사 연계 예제

문제 15번: 0.10 M H₂SO₄ 100 mL를 4.9 g/L로 만들려면 물을 얼마나 추가?
(H₂SO₄ 몰질량 = 98 g/mol)

정답: (C) 100 mL

해설:
1단계: 현재 농도 g/L로 변환
0.10 M = 0.10 mol/L × 98 g/mol = 9.8 g/L

2단계: 희석 배율 계산
9.8 g/L → 4.9 g/L = 2배 희석

3단계: 최종 부피 계산
$C_1V_1 = C_2V_2$
9.8 × 100 = 4.9 × $V_2$
$V_2$ = 200 mL

4단계: 추가할 물의 양
200 mL - 100 mL = 100 mL
황산(H₂SO₄)은 2단계 이온화되므로 1 mol → 2 mol H⁺를 생성합니다!

4단원: 산화-환원 반응

출제 포인트: 산화수 계산, 전자 이동량, 산화제/환원제 식별, 반쪽 반응식

4.1 학습 목표

  • 산화수 계산
  • 산화제/환원제 식별
  • 반쪽 반응식 작성
  • 전자 이동량 계산

4.2 산화-환원 기본 개념

🔸 정의

  • 산화(Oxidation): 전자 잃음 (산화수 ↑)
  • 환원(Reduction): 전자 얻음 (산화수 ↓)
  • 산화제: 자신은 환원되고, 상대를 산화시킴
  • 환원제: 자신은 산화되고, 상대를 환원시킴
암기법: OIL RIG
Oxidation Is Loss (of electrons)
Reduction Is Gain (of electrons)

4.3 산화수 규칙

물질 산화수
원소 (H₂, O₂, Na) 0
단일 이온 (Na⁺, Cl⁻) 전하와 같음
산소 (대부분) -2
수소 (대부분) +1
알칼리 금속 (1족) +1
알칼리 토금속 (2족) +2

4.4 반쪽 반응식

🔸 주요 반쪽 반응

MnO₄⁻ (망가니즈산 이온)
$\\text{MnO}_4^- + 8\\text{H}^+ + 5e^- \\rightarrow \\text{Mn}^{2+} + 4\\text{H}_2\\text{O}$
(5개 전자 얻음 = 환원)
I⁻ (아이오딘 이온)
$2\\text{I}^- \\rightarrow \\text{I}_2 + 2e^-$
(전자 잃음 = 산화)

4.5 모의고사 연계 예제

문제 16번: MnO₄⁻ 1 mol이 I⁻를 몇 mol 산화시키는가?

정답: (E) 5

해설:
1단계: 반쪽 반응식
- $\\text{MnO}_4^- + 8\\text{H}^+ + 5e^- \\rightarrow \\text{Mn}^{2+} + 4\\text{H}_2\\text{O}$
- $2\\text{I}^- \\rightarrow \\text{I}_2 + 2e^-$

2단계: 전자 균형
- MnO₄⁻ 1개 → 5e⁻ 얻음
- I⁻ 1개 → 1e⁻ 잃음

3단계: 몰비 계산
MnO₄⁻ 1 mol이 5 mol 전자를 받으려면
→ I⁻ 5 mol 필요
문제 12번: Zn은 Sn²⁺를 환원시키고, Pb는 안 함 → 결론?

정답: (C) Zinc is a stronger reductor than lead

해설:
- Zn + Sn²⁺ → Zn²⁺ + Sn (반응 O)
- Pb + Sn²⁺ → 반응 없음 (반응 X)

→ Zn이 Sn²⁺를 환원 = Zn이 더 강한 환원제
→ Pb는 반응 안 함 = Pb가 더 약한 환원제

환원력 순서: Zn > Sn > Pb

5단원: 양적 관계와 기체 법칙

출제 포인트: 몰 계산, STP 부피(22.4 L/mol), 한계 반응물, 이상 기체 법칙

5.1 학습 목표

  • 화학 반응식 균형
  • 한계 반응물 계산
  • 기체 부피 (STP에서 22.4 L/mol)
  • 이상 기체 법칙 적용

5.2 몰(Mole)의 개념

1 mol = $6.02 \\times 10^{23}$ 개 (아보가드로 수)

몰 수(mol) = $\\frac{\\text{질량(g)}}{\\text{몰질량(g/mol)}}$

🔸 몰질량 계산

H₂SO₄의 몰질량:
H(1) × 2 + S(32) + O(16) × 4 = 2 + 32 + 64 = 98 g/mol

5.3 기체 법칙

🔸 STP 조건

  • 온도: 0°C (273 K)
  • 압력: 1 atm (101.3 kPa)
  • 1 mol 기체 = 22.4 L

🔸 이상 기체 법칙

pV = nRT

p = 압력 (atm)
V = 부피 (L)
n = 몰 수 (mol)
R = 기체 상수 (0.0821 L·atm/mol·K)
T = 절대온도 (K)

5.4 한계 반응물

🔸 판단 방법

  1. 각 반응물의 몰 수 계산
  2. 반응식의 몰비와 비교
  3. 몰 수 ÷ 계수 값이 작은 것이 한계 반응물

5.5 모의고사 연계 예제

문제 11번: NH₄ClO₄ 11.75 ton + Al 2.70 ton → N₂ 부피는?

반응식:
6NH₄ClO₄ + 10Al → 3N₂ + 5Al₂O₃ + 6HCl + 9H₂O

정답: (D) 6.72×10⁵ L

해설:
1단계: 몰 수 계산
- NH₄ClO₄: $ rac{11.75 imes 10^6 ext{ g}}{117.5 ext{ g/mol}} = 1 imes 10^5$ mol
- Al: $ rac{2.7 imes 10^6 ext{ g}}{27 ext{ g/mol}} = 1 imes 10^5$ mol

2단계: 한계 반응물 찾기
- NH₄ClO₄: $ rac{1 imes 10^5}{6} = 16,667$ (계수로 나눈 값)
- Al: $ rac{1 imes 10^5}{10} = 10,000$ ← 한계 반응물!

3단계: N₂ 생성량
10 mol Al → 3 mol N₂
$1 imes 10^5$ mol Al → $3 imes 10^4$ mol N₂

4단계: 부피 계산 (STP)
V = $3 imes 10^4$ mol × 22.4 L/mol = 6.72×10⁵ L
한계 반응물을 정확히 찾는 것이 핵심! 계수로 나눈 값이 가장 작은 것이 한계입니다.

6단원: 반응 속도와 화학 평형

출제 포인트: 촉매의 역할, 활성화 에너지, 반응 속도 그래프, 온도 단위 변환

6.1 학습 목표

  • 반응 속도에 영향을 주는 요인
  • 촉매의 역할 이해
  • 평형 상태와 르샤틀리에 원리
  • 반응 경로와 활성화 에너지

6.2 충돌 이론

🔸 반응 조건

  • 분자들이 충돌해야 함
  • 충돌 시 활성화 에너지 이상의 에너지 필요
  • 적절한 방향과 각도로 충돌
활성화 에너지 (Ea) = 반응이 일어나는데 필요한 최소 에너지

6.3 반응 속도 영향 요인

요인 효과 이유
농도 증가 속도 ↑ 충돌 빈도 ↑
온도 상승 속도 ↑ 분자 운동 에너지 ↑
촉매 추가 속도 ↑ 활성화 에너지 ↓
표면적 증가 속도 ↑ 접촉 면적 ↑

6.4 촉매(Catalyst)

🔸 촉매의 특징

  • 활성화 에너지를 낮춤 → 반응 속도 증가
  • 반응 전후 자신은 변화 없음
  • 평형 위치는 변하지 않음 (최종 생성물 양 동일)
  • 정반응과 역반응 속도를 똑같이 증가시킴
촉매는 속도만 빠르게 할 뿐, 수율(최종 생성량)은 변하지 않습니다!

6.5 모의고사 연계 예제

문제 13번: H₂O₂ 분해 그래프 - 촉매 유무에 따른 반응

정답: (B)

해설:
촉매가 있는 반응의 특징:
1. 초기 기울기가 더 큼 (빠른 속도)
2. 최종 생성물 양은 동일 (수평선 높이 같음)
3. 더 빨리 평형에 도달

그래프 (B)가 이 조건을 만족!
문제 14번: 이상 기체 V-T 그래프 → p(t)는?

정답: (C)

해설:
이상 기체 법칙: pV = nRT
→ p = $\\frac{nRT}{V}$

그래프 분석:
- V가 일정하고 T가 증가 → p 증가
- V가 증가하고 T가 일정 → p 감소

구간별로 압력 변화를 추적하면 (C)가 정답

6.6 온도 단위 변환

섭씨 → 화씨: $°F = \frac{9}{5} °C + 32$

화씨 → 섭씨: $°C = \frac{5}{9}(°F - 32)$

섭씨 → 켈빈: $K = °C + 273.15$
문제 20번: °C = °F가 되는 온도는?

정답: (A) -40°

해설:
x = $\\frac{9}{5}x + 32$
x - $\\frac{9}{5}x$ = 32
$-\frac{4}{5}x$ = 32
x = -40
